This is a demo page

This is not an exciting website, but shows how you can use your Github repo as a web page host instead of just a remote repository.

Follow these instructions to make your own:

  1. Create a github account, or login to an existing account.
  2. Create a new repository, name it [username].github.io (insert your acccount username).
  3. Make sure the repository is set to
  4. Download Github Desktop or Gitscm
  5. Run Github Desktop.
  6. Clone your newly-created repo to your computer. Make note of the location.
  7. Copy your website code to the local repo clone folder.
  8. In Github Desktop, commit the changes to the repo with a summary comment.
  9. Push your changes to the repo.

If you visit your github.io page, you should see your website.
